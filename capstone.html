Capstone Project

Features

    Capstone Home page
  • Once logged in, the user could add an entry, query the data, or view the live tableau dashboard.
      • Add New Entry Page
  • If the user chose to add entry the user would have a form to fill that would require:
      Age
      Gender
      Education Level
      Job Title
      Years of Experience
      Salary
  • Upon successful insertion into the dynamodb database, an id would be assigned.
      • Query Data Page
  • The user is prompted to enter an id.
  • The respective data will be displayed.
      • View Dashboard Page
  • Above, I included a snippet of one of the charts on the dashboard.
  • Was a live connection between dynamodb database and tableau, utilizing an aws endpoint.
  • Included many different charts and graphs, that as updated by a user, would automatically adjust onto the charts.

      • Technologies

        Pandas
        Utilizes dataframes/series to organize data into queryable tables in python.
        Primarily used for data cleaning.
        Flask
        Used to orcestrate the page routes.
        Tableau Desktop
        Connected to AWS Dynamodb endpoint for live data connection.
        AWS API & Lambda
        Used to communicated the the AWS dynamodb databases which stored user data, as well as statistical data.
        CRUD operations performed to retreieve and insert data.
